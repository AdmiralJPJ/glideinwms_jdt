glideinWMS Installation Guide


If you looked at the description of the glideinWMS, you probably guessed that installing a glideinWMS is a multi-step process.

You will need to install several independent pieces:
  • The glidein WMS Collector and collocated glidein Factory node
  • One or more GCB nodes
  • The glidein pool Collector node
  • One or more scheduler nodes for user submission
  • The VO Frontend
While the documets presented here assume you will install each component on a different machine, there is nothing preventing you from installing all of them on fewer machines, or even on one machine only.
However, the minimum recommended configuration would be composed of 3 nodes:
  • The glidein WMS Collector and collocated glidein Factory node
  • A node containing a GCB server and the VO Frontend
  • A node containing the glidein pool Collector together with the scheduler for user submissions
